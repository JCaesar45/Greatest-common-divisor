# VELORA • Eternal Craft

A hand-crafted, obsessive digital experience built to stop scrollers dead in their tracks.

## Built With
- Pure vanilla JS (no frameworks, no bloat)
- Tailwind via CDN + deep custom CSS
- Canvas particle system tuned by feel
- Glassmorphism + luxury micro-interactions
- Real scroll-driven progress + 3D card tilt

## Why it hits different
Every curve, every easing function, every hover state was considered. 
The particle field reacts to viewport. The hero typography breathes negative space. 
This isn't another Tailwind template - this is intentional craft.

Tested in Chrome, Firefox, Safari. Feels expensive on mobile too.
